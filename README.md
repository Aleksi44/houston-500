Bomber Space
---------

The official Bomber Space Game.


Getting started
-----------

Install deps :
```sh
$ yarn
```
Start web server at http://localhost:8080/ :
```sh
$ yarn start
```
Test your code :
```sh
$ yarn lint
```
Extrude tiles
```sh
tile-extruder --tileWidth 16 --tileHeight 16 --input ./tiles_tmp.png --output ./tiles.png
```
